# Ride Sharing Platform

Uber-like ride hailing system.

## Features

- **Riders** - Request rides, tracking
- **Drivers** - Accept rides, navigation
- **Matching** - Real-time driver matching
- **Pricing** - Dynamic/surge pricing
- **Payments** - In-app payments
- **Ratings** - Driver/rider ratings
- **GPS** - Real-time location

## Quick Start

```bash
docker compose up -d

# Rider App: http://localhost:3000
# Driver App: http://localhost:3001
```
