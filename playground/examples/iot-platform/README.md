# IoT Platform

Internet of Things device management and data platform.

## Features

- **Device Registry** - Register, provision devices
- **MQTT Broker** - Device messaging
- **Data Ingestion** - Sensor data collection
- **Time-series DB** - Historical data storage
- **Rules Engine** - Automated actions
- **Dashboards** - Real-time visualization
- **Alerts** - Threshold notifications

## Quick Start

```bash
docker compose up -d

# Dashboard: http://localhost:3000
# MQTT: mqtt://localhost:1883
```
