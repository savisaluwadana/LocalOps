# API Documentation Generator

Auto-generate API docs from OpenAPI specs.

## Features

- **OpenAPI 3.0** - Swagger spec support
- **SwaggerUI** - Interactive docs
- **ReDoc** - Beautiful docs
- **Mock Server** - API mocking
- **Validation** - Request validation

## Quick Start

```bash
docker compose up -d

# Swagger UI: http://localhost:8080
# ReDoc: http://localhost:8081
# Mock Server: http://localhost:4010
```
