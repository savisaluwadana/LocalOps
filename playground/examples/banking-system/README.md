# Banking/Finance System

Core banking and transactions platform.

## Features

- **Accounts** - Savings, checking, loans
- **Transactions** - Transfers, payments
- **Cards** - Virtual/physical cards
- **KYC** - Identity verification
- **Statements** - PDF generation
- **Notifications** - Transaction alerts
- **Audit Trail** - Full logging

## Quick Start

```bash
docker compose up -d

# Portal: http://localhost:3000
```
