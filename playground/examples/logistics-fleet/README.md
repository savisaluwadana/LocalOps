# Logistics/Fleet Management

Vehicle and shipment tracking system.

## Features

- **Vehicles** - Fleet registry, maintenance
- **Drivers** - Assignments, schedules
- **Shipments** - Create, track, deliver
- **GPS Tracking** - Real-time location
- **Routes** - Optimization, ETA
- **Geofencing** - Zone alerts
- **Reports** - Fuel, distance, utilization

## Quick Start

```bash
docker compose up -d

# Dashboard: http://localhost:3000
# Map View: http://localhost:3000/map
```
