# Healthcare/Clinic Management

Medical practice management system.

## Features

- **Patients** - Registration, records
- **Appointments** - Scheduling, reminders
- **Doctors** - Profiles, availability
- **Prescriptions** - E-prescriptions
- **Billing** - Invoices, insurance
- **Lab Results** - Upload, view results
- **HIPAA Compliant** - Audit logging

## Quick Start

```bash
docker compose up -d

# Portal: http://localhost:3000
```
