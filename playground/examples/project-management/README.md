# Project Management System

Team collaboration and project tracking.

## Features

- **Projects** - Create, archive, templates
- **Tasks** - Kanban, lists, calendar
- **Teams** - Members, roles, permissions
- **Time Tracking** - Log hours, reports
- **Documents** - Wiki, file sharing
- **Comments** - Discussions, mentions
- **Integrations** - GitHub, Slack

## Quick Start

```bash
docker compose up -d

# App: http://localhost:3000
```
