# Analytics Dashboard

Real-time business analytics platform.

## Features

- **Dashboards** - Drag-drop widgets
- **Charts** - Line, bar, pie, heatmaps
- **Data Sources** - PostgreSQL, MySQL, APIs
- **SQL Editor** - Custom queries
- **Scheduled Reports** - Email delivery
- **Alerts** - Threshold notifications
- **Embeds** - Embed charts anywhere

## Quick Start

```bash
docker compose up -d

# Dashboard: http://localhost:3000
# Metabase: http://localhost:3001
```
