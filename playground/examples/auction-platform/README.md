# Auction Platform

Online bidding and auction system.

## Features

- **Auctions** - Create, schedule
- **Bidding** - Real-time bids
- **Auto-bid** - Proxy bidding
- **Notifications** - Outbid alerts
- **Payments** - Winner pays
- **Categories** - Item types
- **Timer** - Countdown

## Quick Start

```bash
docker compose up -d

# App: http://localhost:3000
```
