# Survey/Forms Platform

Form builder and survey collection.

## Features

- **Builder** - Drag-drop form builder
- **Questions** - Multiple types
- **Logic** - Conditional branching
- **Responses** - Collection, export
- **Analytics** - Charts, insights
- **Themes** - Custom branding
- **Embeds** - Embed anywhere

## Quick Start

```bash
docker compose up -d

# Builder: http://localhost:3000
```
