# Video Streaming Platform

Media streaming and content delivery.

## Features

- **Upload** - Video upload, transcoding
- **Streaming** - HLS/DASH adaptive
- **Thumbnails** - Auto-generated
- **Playlists** - User collections
- **Comments** - Engagement
- **Analytics** - View counts, watch time
- **CDN Ready** - Edge caching

## Quick Start

```bash
docker compose up -d

# App: http://localhost:3000
```
