# Social Media Platform

Social networking with feeds, posts, and messaging.

## Features

- **Profiles** - User profiles, followers
- **Posts** - Text, images, videos
- **Feed** - Personalized timeline
- **Messaging** - Direct messages, groups
- **Notifications** - Real-time alerts
- **Search** - Users, hashtags, content
- **Likes/Comments** - Engagement

## Quick Start

```bash
docker compose up -d

# App: http://localhost:3000
```
